Interface

Canta does not use menu, all functions are accessed through buttons and other interface objects.

Apart from the dialogue occasionally used, Canta has only one window that includes all the buttons and displays. The description of this window is divided into several parts for easy reading. These parts correspondign to the main functions are the following:

  • Audio: the microphone and the notes analysis
  • The MIDI player
  • The buttons
  • The main display
