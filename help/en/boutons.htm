The buttons

The main functions of Canta are accessible via buttons, in the upper right, lower left and lower right of the main window:

Description

Starts your default browser (Internet Explorer, Firefox, ...) and displays the first page of the help that you are currently viewing.

Starts your default browser and connects to the site of Canta for you to be aware of new developments.

Setting of Canta main parameters, allows you to select the microphone, the sampling frequency and the MIDI device. See page: Parameters.

Opens a dialog box to select some options like the MIDI instrument to use with the melody or Canta'skin. See page:Options.

Stop playing the microphone and MIDI file, and lets you view everything that has been sung and played so far and from the start, or the last deletion. To return to normal mode, simply press again this button. See page: Viewing notes.

Erarse Cantas's memory and the notes display.

Save what you sing for you to listen and appreciate your performance. To stop recording, simply click again on this button. During recording, a red indicator is shown next to the status of the player, above the stopwatch. See page: Saving the voice.

Complete stop of the program. Beware! The exit is made without confirmation request!
